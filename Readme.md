# Trileuco

## Instruciones

1-Bajar el repositorio.
 git clone https://github.com/enriquepardoaldao/Swapi.git
 
 2-Entrar en carpeta Swapi
 cd Swapi
 
 3-Ejecutar en linux o Mac:
 ./mvnw spring-boot:run
